# Movie Review Sentiment Analysis Dashboard

A Machine Learning web application that analyzes movie reviews and predicts whether the sentiment is Positive or Negative.

## Features

* Movie review sentiment prediction
* Confidence score display
* Interactive dashboard UI
* IMDb movie review dataset
* Machine Learning powered classification

## Tech Stack

* Python
* Flask
* HTML
* CSS
* Scikit-Learn
* Pandas

## Machine Learning

* TF-IDF Vectorization
* Logistic Regression

## Dataset

IMDb Movie Review Dataset (50,000 Reviews)

## Accuracy

89.54%
